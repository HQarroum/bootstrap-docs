> This repository exposes the extracted `docs.css` stylesheet from the Bootstrap documentation for Bootstrap 3 and Bootstrap 4.

## Install

##### Using NPM

```bash
npm install --save bootstrap-docs
```

##### Using Bower

```bash
bower install --save bootstrap-docs
```
